**Want to explore our code or run it locally?**

</div>

```bash
# 🌟 Clone our artistic repository
git clone https://github.com/rishi043/snehkriti-website.git

# 📁 Navigate to the project directory
cd snehkriti-website

# 💻 Open in your favorite code editor
code .

# 🌐 Or simply open index.html in your browser
# Double-click on index.html and watch the magic unfold!
```
